# Directory: 0_SoftwareEnvironment

Description:

This directory is to facilitate researchers who are not experienced with a specific programming language (e.g., C++) or development environment (e.g., Anadonda). 

**0_README-General.md**: provides general information and pointers to (web) resources to get you going. 

**0_README-ProjectSpecific.md**: provides project specific information about the used programming language or development environment.



In addition, the different sub-directories may contain other relevant information to get started (e.g., cheat sheets, tutorials, etc)



Note:

* Not all of these environments might be used in the current project.
